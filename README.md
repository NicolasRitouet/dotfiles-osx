# mac OS X dotfiles  


## Installation
```bash
ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"
brew install git
brew install htop
brew install node
git clone git@github.com:NicolasRitouet/dotfiles-osx.git
cd dotfiles-osx
./install.sh
````
